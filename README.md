QR Code Generator

QR code generator that runs entirely in the browser. No server, no API, no tracking — just open index.html and it works.


Features


Instant QR code generation from any link
Custom QR color and background color
4 size options (small → XL)
Export as PNG or SVG
Last 5 links saved locally in your browser


Usage

Just open index.html in any browser. No install, no dependencies.
